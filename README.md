📈 Professional Trading Analysis Dashboard
An interactive, real-time Financial Data Analysis web application built with Python. This tool helps traders visualize stock trends, track technical indicators, and make data-driven decisions.

📝 Overview
This project is designed to bridge the gap between complex raw financial data and actionable trading insights. Users can search for any ticker (Stock/Crypto), select timeframes, and analyze market momentum using industry-standard technical indicators.

🚀 Key Features
Real-time Data Fetching: Integrated with yfinance API for live market updates.

Technical Indicators: Automatic calculation of:

Moving Averages (SMA/EMA): For trend identification.

RSI (Relative Strength Index): To identify overbought/oversold conditions.

MACD: For momentum and trend reversals.

Interactive Charts: High-fidelity candlestick and line charts using Plotly.

Portfolio Tracking: Quick summary of price changes and volume spikes.

🛠️ Tech Stack
Language: Python 3.x

Web Framework: Streamlit

Data Handling: Pandas, NumPy

Visualizations: Plotly (Interactive), Matplotlib

API: YFinance (Yahoo Finance API)
